A cache consists of a collection of cache lines, which are copies of uniform size lines of memory, and method of storing the order of which the cache lines where added to the cache.
The cache lines will be stored in an array, with their index's stored in a stack with the original memory address.

When memory is requested, the cache will check for the address in the stack.
If it is found, the data will be returned from the array at the index in the stack.
If it is not found, a cache line will be requested from main memory starting at the required address.
If the cache is full, the stack will be popped, removing the most recent cache line.
Then the new cache line will be stored in the array at the old cache line's index, then the index and address will be pushed onto the stack.
The old cache line is written back to main memory at its original location.

When memory is required to written to, the cache will check for the address in the stack.
If its found, the cache line will be written to.
If it is not found, the cache will request the data from memory and cache it using the afore mentioned method.
Once the data is stored in the cache, the cache line will be written to.
